# android-aiyouxi-lab

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面在功能、主题和用途上各不相同，均为独立开发的前端静态资源。仓库以文件归档为主要目的，不针对任何特定域名或网站提供支持。

## 目录说明

```
.
├── index.html          # 仓库首页/导航页（可选）
├── pages/              # 存放所有独立 HTML 页面的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 公共静态资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：各独立 HTML 页面存放于此，每个文件为一个完整的可访问页面。
- `assets/`：页面可能引用的公共样式、脚本或图片资源，按需组织。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开运行。
- 页面之间不存在依赖关系，彼此独立。
- 归档内容可能包含实验性、学习性或功能性页面，不保证所有页面长期可用或内容更新。
- 部分页面可能引用外部资源（如 CDN 库），请确保网络环境允许访问。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件。
3. 或通过 GitHub Pages 等静态托管服务部署后访问（需自行配置）。
